# Developer Outreach

## Dependencies

No dependencies

## Description 

A multichannel developer outreach program designed to grow the Cardano developer community and provide onboarding support to new builders. This deliverable covers three key items: 
- **Live Streams and video tutorials:** The focus of this proposal is not on creating content, but we acknowledge that this is a non-negligible contributor to DevX. So, we'll have a dedicated DevRel working on creating well-timed content and answering all questions about how to develop on Cardano. This will be done both on live streams and, if the subject is worth it, creating well-structured videos. This will serve also to fill the content void for the new tooling we'll create.
- **Build club:** For new teams that commit to build on Cardano but are not experienced enough, we'll provide technical support to reduce the cost for them to build an MVP. The only condition being that all sessions will be recorded and shared publicly so other builders can learn from them.
- **Maintain a Discord channel:** We'll choose a specific channel to engage with developers. This channel will serve both as a catch-all for new developers that are not sure where to go, and as a router to redirect to the specialized Discord servers/channels. For example, if a developer has questions about MeshJS, we'll attempt to answer them and, at the same time, point the developer to MeshJS's Discord server).

## Outcome 

The outcome is that new builders will always have a way to get unstuck when getting started on Cardano. It will also help with retention of users that are at the early stages of exploring if Cardano is a suitable ecosystem to build on top of.
