# Developer Outreach

## Dependencies

Initial outreach (Discord support, answering questions) starts immediately with no dependencies. Full activation — live streams, video tutorials, and Build Club sessions — requires:

- [Developer HUB](/deliverables/developer-hub.md) — content and live streams will reference and explain it
- [cardano-init](/deliverables/cardano-init.md) — demos and tutorials will use it as the primary project entry point

## Description 

A multichannel developer outreach program designed to grow the Cardano developer community and provide onboarding support to new builders. This deliverable covers three key items: 
- **Live Streams and video tutorials:** The focus of this proposal is not on creating content, but we acknowledge that this is a non-negligible contributor to DevX. So, we'll have a dedicated DevRel working on creating well-timed content and answering all questions about how to develop on Cardano. This will be done both on live streams and, if the subject is worth it, by creating well-structured videos. This will also serve to fill the content gap for the new tooling we'll create.
- **Build club:** For new teams that commit to building on Cardano but are not experienced enough, we'll provide technical support to reduce the cost for them to get started. The only condition is that all sessions will be recorded and shared publicly so other builders can learn from them.
- **Maintain a Discord channel:** We'll choose a specific (existing) channel to engage with developers. This channel will serve both as a catch-all for new developers who are not sure where to go and as a gateway to the specialized Discord servers/channels. For example, if a developer has questions about MeshJS, we'll answer them while also pointing them to MeshJS's Discord server.

## Outcome 

The outcome is that new builders will always have a way to get unstuck when getting started on Cardano. It will also help with the retention of users who are at the early stages of exploring if Cardano is a suitable ecosystem to build on top of.
