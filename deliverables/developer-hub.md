# Developer HUB

Create (or adapt) the entry point HUB for Cardano developers. The first option is to align with the Cardano Foundation to use developers.cardano.org. The objective is to have a single, authoritative entry point for developers new to Cardano that consolidates and organizes onboarding content for three key user paths: EVM/Blockchain developer, Web2 developer, and Non-technical interested user. The HUB will include a CI/CD pipeline to maintain working code snippets, ensuring that all examples remain functional and up-to-date as the ecosystem evolves. It will also serve as the central location for design patterns, best practices, and advanced documentation content.
