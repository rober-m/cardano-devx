# Setup CLI

Create a Setup CLI/TUI to easily start dapp projects — a "create-react-app" for Cardano. Users can pick any desired stack combination from the available on-chain and off-chain tooling. This high-level umbrella tool will leverage all identified ecosystem tooling, provide AI/LLM-native commands for scaffolding and development workflows, and use algorithmic ranking to recommend the most appropriate tools based on the developer's context and goals. The CLI will serve as the primary entry point for new developers setting up their first Cardano project, dramatically reducing the time from "I want to build on Cardano" to "I have a running project."
