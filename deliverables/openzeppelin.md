# OpenZeppelin-like Library

Design and implement a library of standardized and reusable smart contracts — both on-chain and off-chain — with an emphasis on DeFi primitives. Inspired by OpenZeppelin's role in the EVM ecosystem, this library will provide battle-tested, ready-to-audit contract implementations that developers can use as building blocks for their applications. The deliverable includes creating the infrastructure and website for the library, and shipping at least 5 ready-to-audit smart contracts covering common DeFi patterns. By providing canonical implementations of common contract patterns, this library will significantly reduce the time and risk involved in building on Cardano.
